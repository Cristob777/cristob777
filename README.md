## What I've built

### 🔹 B2B E-commerce Operations Platform
End-to-end analytics and automation platform for a mid-market B2B distributor handling $2M+ in annual revenue and a 15,000 SKU catalog.

Built on top of Bsale ERP via 10+ REST API endpoints into a Supabase analytical warehouse. Features include:

- **FIFO invoice conciliation** — replaced manual Excel tracking with automated ageing, overdue alerts, and credit blocking logic
- **Triple CRM segmentation** — RFM scoring + commercial tiers + wholesale/retail classification with automated follow-up routing
- **Internal BI agent (LangChain + Claude API)** — natural language interface that lets the team query the data warehouse in plain English
- **Operational dashboards** — sales analytics, inventory tracking, payment and credit monitoring

**Stack:** Next.js 14 · Supabase · TypeScript · LangChain · Claude API · Vercel
**Outcome:** Reporting cycle reduced from 48 hours to under 5 minutes

### 🔹 AI Agents for Import & Customs Operations
Production AI agents for a B2B import operation. Monitor email inboxes, extract structured data from supplier invoices, customs declarations (DIN documents), payment orders, and landed cost calculations into a PostgreSQL database. Multi-stage workflow with audit trail.

**Stack:** Claude API · Node.js · PostgreSQL · email parsing pipeline
**Outcome:** Eliminated manual data entry across import operations lifecycle
